Div dan Span

Div dan Span merupakan tag yang digunakan untuk mengelompokkan tag-tag HTML dan memberikan informasi kepada tag tersebut.
Div dan Span tidak memiliki arti, sampai diberikan Style
Contohnya Header, Navigasi, Main, Copyright, dll.
Property Display defaultnya yaitu:
Div = Block
Span = Inline

Display

Setiap tag pada HTML berada dalam sebuah kotak, properti display pada css mengatur perilaku dari kotak tersebut.
Setiap tag pada HTML memiliki property display default. Dan value nya dapat diubah.
Value dari display ada 4 yaitu:
inline
inline-block
block
none
